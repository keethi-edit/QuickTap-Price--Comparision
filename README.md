# QuickTap - Price Comparison Platform

## Description
QuickTap is a web-based application that allows users to compare product prices from multiple local shops and choose the best option.

## Features
- Compare product prices across shops
- User login and registration
- Add to cart and purchase
- Shop owner product management
- Admin panel

## Technologies Used
- Java (Servlets)
- JSP
- MySQL
- HTML, CSS

## How to Run
1. Import project into NetBeans/Eclipse
2. Configure MySQL database
3. Import db.sql
4. Run on Apache Tomcat server
